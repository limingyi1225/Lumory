最该警惕的"可能已经在 prod 默默坏着"的几条:
- **P0-1 / P0-2**(audio iCloud 写失败仍删本地档)—— **真实数据丢失**,可能已经发生过。9 个 Opus reviewers 全部漏掉,Codex 一次抓出。优先级最高,1-2 行 `do/catch` 修。
- **#1**(axios err 泄漏 `Authorization: Bearer <OPENAI_API_KEY>` 进日志)—— 一行 redact path 修。**OPENAI_API_KEY 可能已经在 PM2 logs 里裸落盘,登录服务器看 `~/server/logs/backend-err.log` 立刻知道**。如果是,要 rotate key。
- **#4**(remote-change `refreshAllObjects` 抹掉用户输入)—— 一行 delete 修。
- **#5**(Localizable.strings 误把 zh/en truncation key 写一致)—— locale routing 全坏掉,实际用户已经在拿错语言的警告字符串。
- **12d**(空 X-App-Secret 静默吞 401 → AI 功能"看起来在但其实全静默挂"):任何新机 clone 后忘了 `cp Lumory.local.xcconfig.sample` 的 dev,都不会发现。
